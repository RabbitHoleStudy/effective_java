## 방식

매주 주말 오전 혹은 오후(재량적) 오프/온라인 함께 본인이 맡은 파트를 발표합니다.

본인이 정리한 파일([본인 아이디].md)를 업로드합니다.
→ 노션에 작성 후 복붙하시면 마크다운 가능합니다.

예제 코드를 작성할 수 있는 부분은 직접 작성해보는 방식으로 하겠습니다.

### 예상 분량

- 1주차 - 2장 전체 = 9
- 2주차 - 3장 전체 + 4장 3개 = 8
- 3주차 - 4장 나머지 8개 = 8
- 4주차 - 5장 전체 = 8
- 5주차 - 6장 전체 = 8
- 6주차 - 7장 전체 = 7 + 리뷰
- 7주차 - 8장 전체 = 8
- 8주차 - 9장 전체 = 12
- 9주차 - 10장 전체 = 9
- 10주차 - 11장 전체 = 7
- 11주차 - 12장 전체 = 6 + 리뷰
